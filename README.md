# Neton Framework 文档

Neton 框架设计规范与用户指南。基于 VitePress 构建。

## 本地开发

```bash
pnpm install
pnpm docs:dev
```

## 构建

```bash
pnpm docs:build
pnpm docs:preview
```

## 目录结构

- `docs/guide/` — 用户指南（教程、最佳实践）
- `docs/spec/` — 规范文档（冻结版 API 规范）
- `docs/archive/` — 历史归档（Kerisy 时期文档）
- `docs/api/` — API 参考（规划中）
